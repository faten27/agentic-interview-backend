# Agentic Interview Council Backend

FastAPI-based backend API for the **Agentic Interview Council** system — an AI-powered interview platform.

Current version: **0.1.0** (early development stage)

## Features
- Start a new interview session
- Submit answers and receive next questions/responses
- Simple health/test endpoints
- Built with FastAPI → automatic interactive documentation

   - Python 3.12.0
   - Git
